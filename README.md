# Torque
Instrumentação de um eixo rotativo para medição de torque em experimento didático
## Resumo
Um dos experimentos didáticos conduzidos no Laboratório de Máquinas de
Fluxo do Departamento de Engenharia Mecânica da Poli requer a medição de torque e
rotação em um eixo que conecta um motor elétrico a uma bomba centrífuga. A medição
confiável dessas grandezas permite caracterizar o desempenho da bomba, avaliando
sua eficiência. Atualmente, esta medição é realizada com um dinamômetro de torção
muito antigo e de difícil leitura. A proposta desse projeto é desenvolver um novo
dispositivo que permita a substituição do dinamômetro antigo. O novo dinamômetro
usará eletrônica de baixo custo para realizar a medida de torque usando extensômetros.
## Estágio atual do desenvolvimento
* eixo instrumentado com extensômetros
* suporte e mancais do eixo construídos
